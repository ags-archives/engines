ASCII code table

This section lists the key codes which can be passed to on_key_press and which keys they represent: 
 1..26     Ctrl+A .. Ctrl+Z
 8         Backspace (Ctrl+H is also Backspace)
 9         Tab (Ctrl+I is also Tab)
 13        Enter (Ctrl+M is also Enter)
 27        Escape
 32        Space
 48..57    '0' key .. '9' key
 65..90    'A' .. 'Z'  (letters are always passed as uppercase)
 359..368  F1 .. F10
 371       Home (including numeric pad 7)
 372       Up arrow
 373       PgUp (including numeric pad 9)
 375       Left arrow
 376       '5' (numeric pad)
 377       Right arrow
 379       End (including numeric pad 1)
 380       Down arrow
 381       PgDn (including numeric pad 3)
 382       Insert
 383       Delete
 433..434  F11 .. F12
Extra codes, which only work with IsKeyPressed (ie. on_key_press is never called with these codes): 
 403       Left shift
 404       Right shift
 405       Left ctrl
 406       Right ctrl
 407       Alt
